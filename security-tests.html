Comment exécuter les tests complets

  1. Lancer un serveur local : python -m http.server 8080 depuis le dossier dashboard/
  2. Ouvrir http://localhost:8080/security-tests.html
  3. Pour les tests de fonctions JS, ouvrir index.html puis coller dans la console :
// Test rapide dans la console du navigateur (depuis index.html)
console.assert(typeof _validateParam === 'function', '_validateParam manquante');
console.assert(_validateParam('<script>') === null, 'XSS accepté !');
console.assert(_validateParam('Démographie') === 'Démographie', 'Valide refusé');
console.assert(typeof escHtml === 'function', 'escHtml manquante');
console.assert(escHtml('<b>') === '&lt;b&gt;', 'escHtml défaillante');
console.log('Tests console : OK');
